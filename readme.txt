Build beautiful price table with ACF: Customize title, content, price, period, link and design to sell your products or services

== Description ==

This plugin introduces a "Table Price ACF Settings" section in the admin panel, allowing you to quickly create a pricing table to sell or present products or services. You can easily add features to your different plans, choose colors and font size and display your price table anywhere with a simple shortcode. It’s totally free.

Just put the shortcode in your post or page and they will work out of the box: **[price_table_acf]**

= Features =
* Simple & Intuitive
* Set features and prices
* Set colors
* Set font size

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Table Price ACF"
3. **Install and Activate** Table Price ACF from your Plugins page
4. **Click** on the new menu item "Table Price ACF Settings" and create your custom table price!

By downloading the ZIP file

1. **Visit** Plugins > Add New
2. **Upload** the ZIP file
3. **Install and Activate** Table Price ACF from your Plugins page
4. **Click** on the new menu item "Table Price ACF Settings" and create your custom table price!
